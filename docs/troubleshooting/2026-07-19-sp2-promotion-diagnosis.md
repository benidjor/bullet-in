월드컵 종료로 열린 SP2 라우팅 1/2순위 비율 재측정을 같은 X 접촉에서 수행했다.

## 1. 증상

— 문제는 승격 (원문 브릿지) 축.

## 2. 진단 방법론

수정 전 원인 기록 원칙에 따라, 저장된 원시 트윗 (mongo) 의 텍스트 · 시각을 그대로 사용해 매칭을 재현했다.

1. mongo `raw_items` 에서 두 인용 트윗의 전체 텍스트 · `created_at` 확보.
2. 기자 타임라인을 어댑터와 같은 경로 (`_scroll_collect` + `_JOURN_JS`) 로 딥 스크롤 (150건 — 운영 depth 25 의 확장판).
3. `match_original_tweet` 의 창 · 겹침 판정을 단계별로 분해:
   창 내 후보 유무 → 창 무시 전체 겹침 점수 상위 → 링크 카드 유무 → 카드 URL 해석 · 도메인 매핑.

## 6. 관찰 (미조치 — 후속 판단)

- **신규 인용 스타일**: afcstuff 가 대괄호 없이 `@handle : "발화 인용"` (멘션 선행) 스타일을 다수 사용 (Romano 인용 등).

  수집 편입 여부는 제품 판단 (발화 인용 vs 보도 인용의 성격 차이) 이 필요해 관찰로 남김.
